4. Preprocessing Calculations / 预处理计算

4.1 Log Growth Matrix (CHEME-5660 Week 5b)

μ_{t,t-1}(r_f) = (1/Δt) · ln(S_t / S_{t-1}) − r_f
where Δt = 1/252, r_f = risk-free rate

Computes excess log returns for all tickers from the JLD2 dataset. Output is a (days × tickers) matrix used for cross-sectional volatility estimation.

4.2 30-Day Rolling Realized Volatility

⚠ SELF-DESIGNED — no direct course reference
σ_RV(t) = std(r_{t-29}, r_{t-28}, ..., r_t) × √252

Annualized rolling volatility using a trailing 30-day window of daily log returns. Computed per ticker per trading day. Floor at 1% to avoid degenerate pricing.

4.3 Dividend Yield

q = (Σ dividends over lookback) / (average price over lookback) / lookback_years

Continuous dividend yield passed to CRR lattice for accurate American option valuation around ex-dividend dates.

4.4 Implied Volatility Calibration (VRP Model)

⚠ SELF-DESIGNED — Entire IV calibration module is original work.
Academic motivation: Carr & Wu (2009) "Variance Risk Premiums"; Bollerslev, Tauchen & Zhou (2009).
Our specific parametric model (VRP × term × skew × volvol) is original.
σ_IV = σ_RV × VRP × (1 + term_adj) × (1 + skew_adj) × (1 + volvol_adj)

VRP multiplier: Safe = 1.15×, Aggressive = 1.25×
term_adj = term_slope × max(0, (30/365 − T) / (30/365)) → shorter options → higher IV/RV
skew_adj = skew_slope × max(0, −moneyness) → OTM puts → higher IV
volvol_adj = vol_of_vol × max(0, σ_RV − 0.30) → high-vol regime → IV rises faster

Data needed for better calibration:

  1. Real options chain data (bid/ask for multiple strikes) → extract actual IV
  2. VIX historical data → make VRP time-varying
  3. CBOE DataShop or OptionMetrics subscription for historical IV surfaces

4.5 Return Correlation Matrix (CHEME-5660 Week 6)

ρ_{ij} = cor(r_i, r_j) on pairwise-complete observations
Used for correlated multi-asset GBM via Cholesky: Z_corr = L · Z_indep, L = cholesky(ρ).L

5. Option Pricing Model / 期权定价模型

5.1 CRR Binomial Lattice — American Options (CHEME-5660 Week 10)

Δt = T / N
u = exp(σ · √Δt), d = 1/u
p = (e^{(r−q)Δt} − d) / (u − d) — risk-neutral probability with dividend yield q

Backward induction (American):
V_j = max(intrinsic_j, e^{−rΔt} × [p · V_{j+1} + (1−p) · V_j])

N = 50 steps. All pricing uses American exercise exclusively — European pricing is never used because the Wheel trades American-style equity options where early exercise matters, especially for puts on dividend-paying stocks.

5.2 Greeks (CHEME-5660 Weeks 11–12)

GreekMethodFormula
Delta (Δ)CRR first levelΔ = (V_u − V_d) / (S·u − S·d)
Gamma (Γ)Central finite diffΓ = (V(S+ε) − 2V(S) + V(S−ε)) / ε²
Theta (Θ)Forward diffΘ = V(T − 1/365) − V(T)
Vega (ν)Central finite diffν = (V(σ+0.01) − V(σ−0.01)) / 0.02

5.3 strike_from_delta — Bisection Search

⚠ SELF-DESIGNED — no course or textbook reference

Input: target |Δ| (e.g., 0.25), current price S, σ, T.
Method: Binary search over K ∈ [0.5×S, 2.0×S]. At each midpoint, compute CRR delta. Converge when K_hi − K_lo < $0.005.

5.4 estimate_implied_vol — Bisection on CRR

⚠ SELF-DESIGNED — CRR-based alternative to BSM IV solver

Input: market option price, S, K, T.
Method: Binary search over σ ∈ [0.01, 3.0]. At each midpoint, price via CRR. Converge when σ_hi − σ_lo < 1e-6.

6. Backtest Engine / 回测引擎

6.1 State Machine per Ticker

SELLING_PUT → (put assigned: S < K) → HOLDING_SHARES → (call assigned: S > K) → SELLING_PUT

Each ticker's Block B has 1–3 ladder slots, each running an independent Wheel cycle. This enables temporal diversification of expiry dates.

6.2 Daily Simulation Loop

StepActionReference
1Deduct daily management fee (0.68% annual / 252)PDF §5
2Compute trailing VaR/ES → throttle if exceeded (VaR > 2.5%, ES > 3.5%)PDF §4, §6
3Compute drawdown_pct → feed into adaptive deltaPDF §3
4Per ticker × slot: deduct borrow cost (0.5% annual on held shares)PDF §7A
5Check dividends → credit Block A + Block B held sharesPDF §2
6Check option expiry → assignment (put ITM) or call-away (call ITM)PDF §2

6.3 Roll Triggers (4 Conditions)

  1. Time-based: DTE in [3, 5] and option is OTM
  2. Premium decay: Current value / original premium < 20% (80% captured)
  3. Moneyness band: |moneyness| > ±5%
  4. Breakeven breach: Price beyond breakeven ± 2%

7. Monte Carlo & Stress Testing / 蒙特卡洛与压力测试

7.1 Geometric Brownian Motion (CHEME-5660 Week 5b)

S_{t+Δt} = S_t · exp[(μ − σ²/2)·Δt + σ·√Δt · Z],   Z ~ N(0,1)

7.2 Correlated Multi-Asset GBM (CHEME-5660 Week 6)

Z_corr = L · Z_indep,   L = cholesky(ρ).L
Each asset j: S_j(t+Δt) = S_j(t) · exp[(μ_j − σ_j²/2)·Δt + σ_j·√Δt · Z_corr_j]

7.3 Regime-Switching GBM

⚠ SELF-DESIGNED — motivated by Varner PDF §7B

2-state Markov chain switching between normal (μ₁, σ₁) and stressed (μ₂, σ₂) regimes with transition probabilities p₁₂ and p₂₁.

7.4 Hidden Markov Model — 2-State (CHEME-5660 Week 13)

EM (Baum-Welch) algorithm:
E-step: Forward (α) and Backward (β) pass → responsibilities γ, transition counts ξ
M-step: Update μ_k, σ²_k from weighted observations; update A from ξ
Output: (μ_normal, σ_normal, μ_stressed, σ_stressed, p_to_stressed, p_to_normal)
